# SenseUI-Spectrum

A visual representation of a dimension and 5 measures

###### FEATURES

###### Line
- Stroke
- Height
- padding

###### X-Axis
- Visibility On/Off

###### Y-Axis
- Visibility On/Off

###### Legend
- Height
- Padding
- Show Labels
- Show Numbers

###### Indicator
- Url of the custom image
- Width
- Height
